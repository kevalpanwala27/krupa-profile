# Krupa Engineering Enterprise - Website

Professional website for Krupa Engineering Enterprise, showcasing industrial equipment manufacturing and engineering services.

## Tech Stack

- **Next.js 15.5.0** - React framework
- **React 19.1.0** - UI library
- **Tailwind CSS v4** - Styling
- **Framer Motion** - Animations
- **TypeScript** - Type safety

## Getting Started

### Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

### Build for Production

```bash
# Build static website
npm run build
```

The static files will be generated in the `out/` folder, ready for deployment.

## Deployment

This is a **static website** that can be deployed to any static hosting service.

See `STATIC_DEPLOYMENT.md` for detailed deployment instructions to Hostinger.

## Project Structure

```
├── src/
│   ├── app/          # Next.js app router pages
│   ├── components/   # React components
│   └── data/         # Data files
├── public/           # Static assets (images, etc.)
└── out/              # Build output (generated)
```
